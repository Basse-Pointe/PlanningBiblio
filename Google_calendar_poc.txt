Google authentication with service account:

Procedure: https://developers.google.com/admin-sdk/directory/v1/guides/delegation

1) Create an app at https://console.developers.google.com/
2) Add a service account for this app (Role: Utilisateur du compte de service) at https://console.cloud.google.com/iam-admin/serviceaccounts
3) Enable G Suite Account Domain-Wide delegation
4) Get json file and client ID
5) Put json file in PLB (see putenv('GOOGLE_APPLICATION_CREDENTIALS=path')
6) In admin, add the following scope to client id: https://www.googleapis.com/auth/calendar.readonly
   (at https://admin.google.com/AdminHome?chromeless=1#OGX:ManageOauthClients)
